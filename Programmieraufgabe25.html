Aufgabenstellung:

Schreiben Sie ein Makro infix, mit dem Sie die Addition zweier Zahlen in Infix-Schreibweise errechnen lassen können. Anwendungsbeispiel: (infix 1 + 2)

Lösungsansatz:

Die Aufgabe wurde zusammen mit Sebastian Espei erarbeitet.
Als erstes wird das erste Element der Liste mit allen Argumenten entfernt. Die Rest-Liste wird in die Variable 'args' gespeichert. Daraufhin wird die 'datum->syntax' Funktion aufgerufen welche die restlichen Elemente entgegen nimmt. Jedoch sind diese noch in der falschen Reihenfolge. Deswegen wird zuerst das Element mit dem Index 1, dann mit dem Index 0 und dann zuletzt das Element mit dem Index 2 übergeben. Somit wird aus der Semantik '1 + 2' die ausführbare Syntax '+ 1 2'.
