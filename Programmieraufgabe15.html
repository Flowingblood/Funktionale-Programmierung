Aufgabenstellung:

Schreiben Sie eine Funktion my-length, die die Länge einer Liste zurückgibt und benutzen dazu foldr oder foldl.

Lösungsansatz:

Bei der Implementierung wird die Funktion my-length mit einer inneren Funktion versehen welche zwei Übergabeparameter haben
muss, da die foldr/l Funktion eine Prozedur entgegennimmt welche zwei Parameter hat.
Der erste Parameter 'p0' wird hierbei gänzlich ignoriert und es wird nur der zweite Parameter 'counter' verwendet, welcher
sich in jedem Funktionsaufruf inkrementiert. Somit wird diese innere Funktion 'my-length-inner', welche den Counter
inkrementiert, so oft aufgerufen, wie die Liste an Elementen hat. Somit wird die Länge der Liste ermittelt.
