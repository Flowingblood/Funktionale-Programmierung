Was sind Funktionen höherer Ordnung?

Ein Funktion der höheren Ordnung kann eine Funktion entgegennehmen oder eine Funktion als Rückgabewert zurückgeben. Das Nutzen von Funktionen höherer Ordnung ist in Racket implementiert, aber in anderen Programmiersprachen nicht möglich. In Java ist dies erst möglich mit der Einführung der Lambda Ausdrücke. Der Anwendungsbereich dieser Möglichkeit ist besonders mit Funktionen wie das Currying, Fold, Map und Compose.


Currying

Beim Currying wird eine Funktion mit mehrere Argumenten umgewandelt in eine Reihenfolge von Funktionen mit genau einem Argument. Dies wird umgesetzt, indem die Funktion eine Anzahl n Argumente erwartet. Nun wird das Argument n von der Funktion verarbeitet und gibt eine Funktion zurück, welche n-1 Argumente erwartet. Dies wird iterativ/rekursiv so lange ausgeführt, bis keine Argumente mehr erwartet werden.


Fold

Bei der Fold (R/L) Funktion wird als Übergabe eine anzuwendende Funktion erwartet, ein initial Wert und eine Liste auf, welche die Funktion angewendet wird. Der Unterschied zwischen dem R und L bei Fold ist die Ausführung auf die Liste, ob diese von rechts oder von links startet.


Map

Die Map Funktion gehört zu den simpleren Funktionen, diese nimmt eine Funktion und eine Liste entgegen, hierbei wird die Funktion auf jedes Element aus der Liste einzeln angewendet.


Compose

Die Compose Funktion nimmt eine unbegrenzte Anzahl an Funktionen entgegen und kombiniert diese zu einer einzigen. Diese werden hierbei von rechts nach links ausgeführt. Somit wenn die Compose Funktion mit den Argumenten („func1“, „func2“, „func3“) aufgerufen wird, wird zuerst die Funktion func3 dann func2 und zuletzt func1 aufgerufen.
