Aufgabenstellung:

Gegeben sei die folgende Funktion in Typed Racket:
(: bar (-> (U Integer Boolean String) Integer))
(define (bar x)
(cond
((number? x) (string-length (number->string x)))
(else (string-length x)))) Wenn man diese Funktion versucht zu kompilieren, erhält man eine Fehlermeldung vom Type CHecker. Erläutern Sie die Fehlermeldung und korrigieren Sie die Funktion.

Lösung:

Der generische Typ U kann ein Integer, Boolean oder ein String sein. Jedoch wird in der Cond bedingung erstmal auf eine Number geprüft, welche dann in einen String umgewandelt wird und die Länge dieses Strings wird zurück gegeben. Nun kann U noch ein boolean oder ein String sein und es ist nicht möglich die Funktion "string-length" auf ein Boolean anzuwenden. Somit ist die Lösung, vorher zu Prüfen ob es ein Boolean ist.
