Aufgabenstellung:

Schreiben Sie eine Funktion sum-my-list, die eine Liste mit Zahlen erhält und diese aufaddiert.

Lösungsansatz:

Es wird eine Summenvariable mit dem Wert '0' initialisiert, daraufhin wird über jedes Element aus der Liste iteriert.
In jeder Iteration wird die Summenvariable neu definiert, indem der momentane Wert der Variable mit dem Element aus der
Liste addiert wird. Diese Summenvariable ist der Rückgabewert.
