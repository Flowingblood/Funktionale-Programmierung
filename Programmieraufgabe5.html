Aufgabenstellung:

Schreiben Sie eine Funktion my-find, die eine Liste lst und ein Argument x erhält und mittels equal? prüft, ob das Element in der Liste enthalten ist. Die Funktion soll #t antworten, wenn dem so ist, oder "#f" wenn nicht.

Lösungsansatz:

Es wird eine boolsche Variable angelegt welche anfangs 'false' ist. Daraufhin wird über jedes Element in der Liste iteriert.
In jeder Iteration wird geprüft ob, dass momentane Objekt mit dem übergebenen Objekt übereinstimmt.
Wenn dies der Fall ist, wird die anfangs angelegt boolsche Variable 'true' gesetzt und am ende der Schleife zurückgegeben
