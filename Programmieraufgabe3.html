Aufgabenstellung:

Schreiben Sie eine Funktion, die eine natürliche Zahl n erhält und eine Liste der natürlichen Zahlen von n bis 1 erzeugt.

Lösungsansatz:

Eine For-Schleife startet die Iteration bei 1 bis zum Element n.
In jeder Iteration wird die Liste modifiziert, indem jeweils ein Element vorne an die Liste angehängt wird.
