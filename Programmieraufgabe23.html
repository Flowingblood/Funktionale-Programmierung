Aufgabenstellung:

Schreiben Sie ein map und ein foldr in Typed Racket.

Lösungsansatz:

Die Aufgabe wurde zusammen mit Sebastian Espei erarbeitet.
Es wurde die my-reverse Funktion aus einer davorigen Aufgabe kopiert um eine Liste zu umzukehren. Um die 'foldr' Funktion in typed Racket umzusetzen muss über dem Funktionskopf definiert werden, was übergeben werden soll. Hierbei wird eine generischer Datentyp 'T' erstellt. Dieser kann alles sein. Als erster Parameter wird eine Funktion erwartet welche zwei Parameter hat welche T sind und die Funktion muss T zurückgeben. Dann folgt ein T Parameter welcher hierbei der init Wert ist und Liste vom Datentyp T welche von rechts nach links durchlaufen wird. Rückgabe ist hierbei auch wieder vom Typen T. Der einfachheit halber wurde die Liste umgekehrt, sodass man in der Funktion von links nach der rechts die Liste abarbeiten kann, aber trotzdem die Ausführungsreihen folge beim anwenden von 'foldr' von rechts nach links durchläuft. In der For-Schleife welche die Liste durchläuft, werden die letzten beiden Werte in der Liste in eine Variable gespeichert. Daraufhin werden die ersten beiden Elemente aus der Liste entfernt. Die beiden gespeicherten Werte werden in die übergebene Funktion gegeben und von rechts an die Liste hinzugefügt. Diese Iteration wird so lange durchlaufen wie die Länge der Liste-1 ist. Zuletzte wird die Liste welche hierbei übrig bleibt, welche nurnoch einen Wert hat mit car in diesen einzelnen Wert transformiert.
Die Map Funktion wird mit zwei generischen Datentypen 'T' und 'G' eingeleitet. Hierbei ist der erste Parameter eine Funktion welche einen Parameter vom Typ 'T' entegegen nimmt und 'G' zurück gibt. 'T' und 'G' können hierbei alle Datentypen einnehmen. Daraufhin wird eine Liste vom Typen 'T' übergeben auf welche dann die Funktion angewendet wird und eine Liste vom Typen 'G' zurückgegeben. In der Map Funktion wird eine leere Liste erzeugt. Nun wird über die übergebene Liste iterriert und in die leere Liste jedes Element aus der übergebenen Liste hinzugefügt nachdem die übergebene Funktion darauf angewendet wurde. Diese Liste wird nun zurück gegeben und ist somit eine Liste vom Typen 'G'.
