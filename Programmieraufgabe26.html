Aufgabenstellung:

(Knifflig) Erweitern Sie das Makro derart, dass Sie belebige Additionen und Subtraktionen vornehmen können. Beispiel: (infix 1 + 2 - 4 + 5).

Lösungsansatz:

Die Aufgabe wurde zusammen mit Sebastian Espei erarbeitet.
Als erstes wird das erste Element der Liste mit allen Argumenten entfernt. Die Rest-Liste wird in die Variable 'args' gespeichert. Daraufhin wird eine Summen-Variable initialisiert, in welche die Summe aller Zahlen gespeichert wird. Daraufhin durchläuft eine For-Schleife die args-Liste bis zum länge von args / 2 gerundet - 1 ((round (n/2))-1). Daraufhin wird wie zuvor auch das zweite, das erste und das dritte Element genommen, verrechnet und die Summen-Variable gespeichert. Daraufhin werden die ersten 3 Elemente mit "cdddr" aus der Liste entfernt und die Summen-Variable vorne an die Liste angehängt. Zuletzt wird die Summen-Variable zurückgegeben.
