Aufgabenstellung:

Schreiben Sie eine Funktion square-my-list, die eine Liste mit Zahlen erhält und eine neue Liste der Quadrate dieser Zahlen zurückliefert.

Lösungsansatz:

In einer For-Schleife wird über jedes Element aus der Liste iteriert. In jeder Iteration wird in einer zuvor leeren Liste
ein Element hinzugefügt, welches das momentan iterrierte Element mit sich selbst multipliziert wird.
