Aufgabenstellung:

Schreiben Sie eine Funktion repeated, die eine Funktion f und eine natürliche Zahl n erhält und dann f n-mal ausführt.
(Hinweis: Verwenden Sie die im Kurs bereits vorgestellten Standardfunktionen höherer Ordnung!) Beispiel:
((repeated sqr 2) 3)
81

((repeated add1 10) 1)
11

Lösungsansatz:

Die erste innere Funktion wird am Ende der gesamten Funktion zurückgegeben. Der Inhalt der zurückgegebenen Funktion ist folgender:
Es wird eine weitere innere Funktion erzeugt, welche zwei Parameter hat. Daraufhin wird geprüft ob der Ausdruck 'r == 0' wahr ist.
Wenn der Ausdruck 'false' ist, wird die zweite innere Funktion aufgerufen, wobei 'r' um eins dekrementiert wird.
