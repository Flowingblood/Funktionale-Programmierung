Aufgabenstellung:

Schreiben Sie eine Funktion stream->list, welche die nächsten n Elemente eines Stroms in einer Liste zurückgibt.

Lösungsansatz:

Die Aufgabe wurde zusammen mit Sebastian Espei erarbeitet.
Zuerst wird der Stream in eine Liste umformatiert und dann mit der for/stream Schleife über jedes Element bis zum n-tn iterriert, daraufhin wird dann die Referenz auf das Objekt als Liste zurückgegeben
